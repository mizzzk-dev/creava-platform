# 環境変数（WordPress 移行版）

## frontend
- `VITE_CMS_PROVIDER` (`strapi` | `wordpress`)
- `VITE_WORDPRESS_API_URL`
- `VITE_STRIPE_PUBLISHABLE_KEY`

## wordpress
- `STRIPE_SECRET_KEY`
- `STRIPE_WEBHOOK_SECRET`
- `STRIPE_DEFAULT_CURRENCY`
- `CREAVA_FRONTEND_URL`
- `CREAVA_ALLOWED_ORIGINS`

## wordpress production hardening（2026-04-25 追加）
- `CREAVA_ALLOWED_ORIGINS`（許可origin host。カンマ区切り）
- `CREAVA_PUBLIC_RATE_LIMIT` / `CREAVA_PUBLIC_RATE_WINDOW_SEC`（公開RESTレート制限）
- `CREAVA_DISABLE_XMLRPC`（既定: `true`）
- `CREAVA_ENABLE_SECURITY_HEADERS`（既定: `true`）
- `CREAVA_MEDIA_ALLOWED_MIME_TYPES`（アップロード許可MIME）
- `CREAVA_MEDIA_MAX_UPLOAD_BYTES`（アップロード上限）
- `CREAVA_SECURITY_AUDIT_LOG`（security監査ログ有効化）
- `WORDPRESS_PREVIEW_SECRET`（preview verify）
- `WORDPRESS_HEADLESS_JWT_SECRET`（headless bearer JWT検証）
- `STRIPE_WEBHOOK_SECRET`（Stripe webhook署名検証）
