# UI Critique チェックリスト

## 1. 体験バランス
- [ ] main がブランド入口として成立している
- [ ] store/fc への導線が自然で、押し売り感がない
- [ ] Contact/Request 導線が埋もれていない

## 2. 可読性
- [ ] 見出し階層が明確
- [ ] 本文の行長・行間が過密でない
- [ ] 背景と文字のコントラストが十分

## 3. 操作性
- [ ] キーボードで主要導線へ到達できる
- [ ] focus 可視性が維持される
- [ ] CTA が状態（通常/hover/disabled）で判別可能

## 4. テーマ/言語
- [ ] light/dark どちらでも破綻しない
- [ ] ja/en/ko で文言折返しが崩れない

## 5. AIテンプレ感の排除
- [ ] 使い回し感の強いレイアウトを乱用していない
- [ ] 意味のない装飾グラデーションを多用していない
- [ ] 画面の主語（作品/購入/会員）が明確
