# ブランド・パーソナリティ定義

## コア
- 静かだが芯が強い
- 作品と活動で信頼を積み上げる
- 商売色よりも体験価値を先に示す

## トーン
- 文体: 端的、誠実、煽らない
- 造形: 余白を活かす、秩序を保つ
- 画像: 作品・人物を主役にし、装飾は引き算

## サイト別の話し方
- **main**: 世界観 + 信頼（実績、最新情報、依頼導線）
- **store**: 購入判断の不安除去（在庫、配送、返品、FAQ）
- **fc**: 会員メリットの具体化（限定性、継続性、マイページ）

## CTA設計
- main: Contact / Request を最優先。
- store: 商品一覧・ガイド・FAQ のセットで安心遷移。
- fc: Join / Login / MyPage を状態に応じて出し分け。
