# デザイン原則（durable）

## 目的
`mizzz.jp` をブランド全体のハブとして維持しながら、`store.mizzz.jp` と `fc.mizzz.jp` を独立体験として成立させる。

## 役割分担
- **main (`mizzz.jp`)**: ブランド入口。About/Works/News/Contact で信頼形成。
- **store (`store.mizzz.jp`)**: 購買判断を支える導線（商品・ガイド・FAQ・法務）。
- **fc (`fc.mizzz.jp`)**: 会員継続を支える導線（Join/Login/MyPage/限定コンテンツ）。

## UI原則
1. **余白先行**: 先に情報階層を作り、装飾は後から足す。
2. **タイポ一貫**: 見出し・本文・補助情報のスケールを固定する。
3. **Hero明確化**: 1画面1メッセージ。CTAは主1・副1まで。
4. **導線の自然性**: main から store/fc へ文脈を持って遷移させる。
5. **過剰演出回避**: 動きは理解補助に限定し、可読性を優先。

## Light / Dark 原則
- 片テーマのみで成立する配色は禁止。
- コントラスト不足、半透明重ねすぎ、背景依存の文字色を避ける。
- hover/focus/disabled の状態差を両テーマで確保する。

## 多言語（ja/en/ko）で崩さない原則
- 文言長差を前提にコンポーネント幅を決める。
- 固定高さ UI は最小限にし、改行時の崩れを許容する。
- CTA ラベルは翻訳後の意味差（強さ）を揃える。

## AIテンプレUI抑制チェック
- どの画面にも同じ「カード + グラデ + 3カラム」を機械的に適用していないか。
- セクション見出しが汎用語（"Amazing", "Next Gen" 等）に寄っていないか。
- 体験の主語（ブランド/商品/会員）に応じた情報密度になっているか。

## やってよい演出
- スクロール連動の微小フェード。
- CTA 周辺の軽いモーション。
- ブランド写真・作品を活かす余白設計。

## やりすぎ禁止
- 常時ループする強いアニメーション。
- 背景ノイズで可読性を落とす処理。
- EC訴求を main 全面に押し出す構成。
